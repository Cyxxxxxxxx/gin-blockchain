# gin-blockchain
A minimalist blockchain project based on the Gin framework

# Gin-blockchain
- ### Web services provided by Gin framework

- ### Blockchain section:
  - ##### Three funtions are encoded
    - ```go
      GenesisBlock() 
      ```
      <!--创世区块-->				

    - ```go
      WriteBlock()
      ```
      <!--写入区块-->

    - ```go
      ShowBlockchain
      ```
      <!--查看区块链-->
